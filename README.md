# Unemployment Analysis in India

This project analyzes unemployment trends in India using Python data analysis and visualization libraries.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Missing Value Handling
- Date Conversion
- Statistical Analysis
- Unemployment Trend Visualization
- Region-wise Analysis
- Correlation Heatmap
- Monthly Trend Analysis

## Dataset
Dataset used:
`Unemployment in India.csv`

## Visualizations Included
- Unemployment Rate Over Time
- Average Unemployment by Year
- Top Regions with Highest Unemployment
- Correlation Heatmap
- Monthly Unemployment Trend

## How to Run

```bash
python unemployment_analysis.py
